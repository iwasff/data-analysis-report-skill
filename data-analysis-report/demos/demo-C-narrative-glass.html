What

什么是 Data Analysis
Report Skill？

不是一份报告，是一套生成报告的方法论——输入数据，输出完整 HTML 报告，平均 30 分钟，可复用可追溯

DIMENSION 01
Mode
报告模式
Analysis（数据分析侧边栏）或 Narrative（全屏叙事 Slide），决定整体布局逻辑

2

DIMENSION 02 · 03

+ UI

DIMENSION 04 · 05
Type
+ Motion

3 种字体风格（serif/sans/mono）× 3 级动效（subtle/standard/expressive）

02 / 07
Architecture

三层解耦架构
消除 AI 幻觉

计算与判断完全分离，Python 只做数学，Claude 只做洞察——结论可追溯，数字可验证

LAYER 01 · 定义层
PRD 字段定义
product_requirements.md
指标口径、业务逻辑、分母定义——用自然语言写成，任何人接手都能对齐，整套体系的「单一真理源」
结构化定义 · 可追溯
→
LAYER 02 · 计算层
Python 纯计算
analyze.py → data.json
只做数学，不做判断，输出结构化 JSON。同样输入永远同样输出——可 diff，可单独验证，出错定位到行
确定性计算 · 无幻觉
→
LAYER 03 · 洞察层
Claude 报告生成
JSON → HTML Report
读取计算好的 JSON，生成带图表、带行动建议的完整 HTML 报告。Claude 做它擅长的——语言与判断，不做数学
洞察 · 可视化 · 行动

Workflow

从数据到报告
三步完成

STEP 01
Phase 0 选风格
Claude 逐轴询问 5 个维度偏好，输出 Style Config 块。一次配置，可复用到后续所有同类报告
~3 min
→
STEP 02
Phase 1 准备数据
上传原始数据（CSV/Excel/JSON），Claude 读取数据结构，规划报告章节，确认后开始生成
~5 min
→
STEP 03
Phase 2 生成报告
按 Config 调用对应维度的代码库，生成自包含单文件 HTML，含图表、洞察、行动建议
~20 min
