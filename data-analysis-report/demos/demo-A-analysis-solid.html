Solid 卡片的优势
明确的色彩边界让层次感直观，不依赖 backdrop-filter，兼容性最佳。Warm-Green 的深绿/浅卡片对比是 Solid UI 的经典组合
Serif 字体的适用场景
Playfair Display 大标题搭配 Noto Serif SC 正文，传达专业质感。适合内部月报、管理层 Review、需要「报告感」的场景

Standard 动效特点
Scroll reveal 触发元素入场，ECharts 图表自带入场动画，counter 计数器。不使用粒子背景，专注内容，加载更快
适合 Standard 的场景
内部汇报、数据共享、需要快速浏览的场景。动效存在但克制——用于引导注意力，而非展示动效本身

地图 · Warm-Green 配色
数据色用琥珀 (#D0986A)，空白区用米白 (#EDE8C0)，符合整体暖色调，没有刺眼的饱和红绿对比
地图适合的数据类型
有地理维度的数据：国家/地区销售、退货率分布、用户来源、合规要求差异。地图比数据表更直观
使用注意事项
世界地图需要加载 JSON 数据（约 400KB），异步加载，网络较慢时建议提供加载占位态，避免空白
